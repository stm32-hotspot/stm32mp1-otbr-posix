Purpose

The X-LINUX_OTBR_POSIX is a linux Package running on STM32MPU and tested on STM32MP157F-DK2 and STM32MP135F-DK2. Openthread Border Router work with Radio Co Processor (RCP) STM32WB55 through serial connection (FTDI) between STM32MPU and STM32WB. This recipes provide all packages needed to compile ot-br-posix "https://github.com/openthread/ot-br-posix" on stm32mpu. Related information and documentation:
  • STM32 MPU Getting Started
